数据库的简单介绍 以及 mysql 的安装 与 启动 知识

数据库简介

数据库就是一个帮助我们管理 数据的 软件

当然管理数据 你也可以 用 excel ，但是 MySQL 相对而言 肯定是 相对 有很多优势，后面会介绍

数据库的分类

  • 关系型数据库
    1. 有约束
    2. 基于 硬盘 存储
    3. 具体软件： MySQL SQLServer(微软 c# .net) Oracle（甲骨文 java）SQLlite DB2 mariadb pgsal
  • 非关系型数据库
    1. 没有约束 （key -> value 的形式）
    2. 基于内存 存储
    3. 具体软件： mongodb redies Memcache

MySQL 数据库安装

mysql 分为 服务端 socket 与 客户端client

  • 下载地址

  • zip 包下载

    直接解压就好了

  • exe文件 或则 msi 文件

    需要点击 一步一步的安装

  • 目录结构

    • bin 目录

      mysqld : 启动服务端

      mysql ：客户端连接服务器

    • data 目录

      存放数据文件

  • 具体安装

    • mac

  • window （此处以 zip 为列）

    1. 解压

    2. cd 到 bin 目录

    3. mysqld (启动mysql 服务)

    4. mysql -uroot -p 回车即可

    5. 此时可以进入 mysql 的命令行

    6. 环境变量的配置
